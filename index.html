Payment Gateway Configuration

Razorpay Configuration (UPI/Cards) Not Configured

Razorpay supports UPI, cards, net banking, and wallets (popular in India). Get your keys from Razorpay Dashboard

How to get Razorpay API Keys:

  • Go to Razorpay.com and sign up
  • Complete KYC verification
  • Go to Settings → API Keys
  • Generate and copy the Key ID and Key Secret
  • Use test keys for testing, live keys for production

Webhook Configuration

Webhooks notify your server when payments are completed. Set up webhook endpoints in your payment provider dashboards.

Webhook Endpoints to Configure:

  • Razorpay: payment.captured

Security Settings

Security Best Practices:

  • Use strong, unique passwords
  • Never share API keys publicly
  • Use test keys during development
  • Enable 2FA on payment provider accounts
  • Regularly rotate API keys
